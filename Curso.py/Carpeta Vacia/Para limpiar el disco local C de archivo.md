# Para limpiar el disco local C de archivos basura y residuos, puedes usar las herramientas integradas de Windows:

1. **Liberador de espacio en disco**:
    - Pulsa `Win + S`, escribe "Liberador de espacio en disco" y ejecútalo.
    - Selecciona la unidad C y marca las casillas de archivos a eliminar.

2. **Eliminar archivos temporales manualmente**:
    - Pulsa `Win + R`, escribe `%temp%` y elimina los archivos de esa carpeta.
    - Repite con `C:\Windows\Temp`.

3. **Desinstalar programas innecesarios**:
    - Ve a `Panel de control > Programas > Desinstalar un programa`.

4. **Antivirus**:
    - Instala un antivirus confiable (Windows Defender, Avast, etc.) y realiza un escaneo completo.

5. **Herramientas de terceros** (opcional):
    - Puedes usar programas como CCleaner para limpiar archivos basura (descárgalo solo desde su web oficial).

No elimines archivos del sistema ni carpetas que no reconozcas para evitar problemas.
